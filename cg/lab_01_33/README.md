Задача №33: На плоскости дано множество точек. 
Найти такой треугольник с вершинами в этих точках у которого угол, 
образованный медианой и биссектрисой исходящими из одной вершины, минимален.
Вывести изображение в графическом режиме. 
Примечание: треугольник нарисован красным, медина - синим, биссектриса - зеленым.
