\ssr{ЗАКЛЮЧЕНИЕ}


Цель работы достигнута: были исследованы методы рекомендательных систем. При выборе методов решения задачи рекомендательных систем необходимо определить наличие и возможность использования данных о товарах, пользователях, предпочтении пользователей, а также уровень персонализации и необходимость решения основных проблем рекомендательных систем.

В ходе выполнения работы все поставленные задачи были выполнены:
\begin{enumerate}
	\item проведён анализ предметной области;
	\item формализована задача рекомендательной системы;
	\item перечислены существующие методы решения задачи;
	\item сформулированы критерии сравнения методов решения;
	\item проведён сравнительный анализ методов решения на основе сформулированных критериев.
\end{enumerate}
