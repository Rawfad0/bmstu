\ssr{ЗАКЛЮЧЕНИЕ}

В ходе выполнения курсовой работы поставленная цель была достигнута: разработано программное обеспечение маркетплейса, которое позволяет продавцам размещать предложения о продаже товаров, а покупателям находить интересующие товары и покупать их.

Были выполнены следующие задачи: 
\begin{enumerate}
	\item проведен анализ предметной области и существующих баз данных;
	\item спроектированы база данных и программное обеспечение;
	\item реализованы спроектированные база данных и программное обеспечение;
	\item проведено исследование разработанной базы данных.
\end{enumerate}
