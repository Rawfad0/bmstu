\ssr{ЗАКЛЮЧЕНИЕ}


В ходе выполнения курсовой работы поставленная цель была достигнута: разработка программного обеспечения для симуляции LiDAR и построение 3D-карты сцены.

Были выполнены следующие задачи: 
\begin{itemize}
	\item описаны объекты и выбрать модель их предсатвления;
	\item изучены и провести анализ существующих алгоритмов компьютерной графики; \item выбраны наиболее подходящие алгоритмы для реализации программы;
	\item спроектировано программное обеспечение, предоставляющее пользователю необходимые функции;
	\item реализовано спроектированное программное обеспечение;
	\item проведен сравнительный анализ.
\end{itemize}
