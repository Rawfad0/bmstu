# Тесты для лабораторной работы №03_02_01
## Входные данные
Целые m, n
Последовательность m*n целых чисел
## Выходные данные
Последовательность (m-1)*(n-1) целых чисел 
## Позитивные тесты:
- 01 - 2х2;
- 02 - все элементы имеют одинаковую сумму цифр;
- 03 - несколько элементов с одинаковой минимальной суммой цифр.

## Негативные тесты:
- 01 - пусто;
- 02 - не является числом (строка);
- 03 - только m;
- 04 - n == 0;
- 05 - m > 10;
- 06 - элементы не введены;
- 07 - количество элементов меньше заданного;
- 08 - получится пустая матрица.
