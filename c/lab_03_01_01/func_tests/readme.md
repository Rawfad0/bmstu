# Тесты для лабораторной работы №03_01_01
## Входные данные
Целые m, n
Последовательность m*n целых чисел
## Выходные данные
Последовательность m целых чисел 
## Позитивные тесты:
- 01 - единственный элемент;
- 02 - нечетное количество элементов в строке;
- 03 - четное количество элементов в строке.
## Негативные тесты:
- 01 - пусто;
- 02 - не является числом (строка);
- 03 - только m;
- 04 - n == 0;
- 05 - m > 10;
- 06 - элементы не введены;
- 07 - количество элементов меньше заданного.
