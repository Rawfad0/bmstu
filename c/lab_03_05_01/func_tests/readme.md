# Тесты для лабораторной работы №03_05_01
## Входные данные
Целые m, n
Последовательность m*n целых чисел
## Выходные данные
Последовательность m целых чисел 
## Позитивные тесты:
- 01 - количество чисел, сумма цифр которых больше 10, равно 1;
- 02 - количество чисел, сумма цифр которых больше 10, равно 6;
- 03 - количество чисел, сумма цифр которых больше 10, равно 2;
- 04 - количество чисел, сумма цифр которых больше 10, равно 3;
- 05 - отрицательные числа.
## Негативные тесты:
- 01 - пусто;
- 02 - не является числом (строка);
- 03 - только m;
- 04 - n == 0;
- 05 - m > 10;
- 06 - элементы не введены;
- 07 - количество элементов меньше заданного;
- 08 - в матрице нет чисел, сумма цифр которых больше 10.
