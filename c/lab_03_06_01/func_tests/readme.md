# Тесты для лабораторной работы №03_06_01
## Входные данные
Целые m, n
## Выходные данные
Последовательность m*n целых чисел 
## Позитивные тесты:
- 01 - 1x1;
- 02 - 2x2;
- 03 - 3x3;
- 04 - 4x4;
- 05 - 10x10.
## Негативные тесты:
- 01 - пусто;
- 02 - не является числом (строка);
- 03 - только m;
- 04 - n == 0;
- 05 - m > 10.
