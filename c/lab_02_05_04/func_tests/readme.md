# Тесты для лабораторной работы №02_05_02
## Входные данные
Целое n
Последовательность n дробных чисел a
## Выходные данные
Целое число counter
## Позитивные тесты:
- 01 - все элементы уникальные;
- 02 - все элементы одинаковые;
- 03 - случайные элементы.
## Негативные тесты:
- 01 - не является числом (строка);
- 02 - n > 10;
- 03 - количество введённых чисел не совпадает с n;
- 04 - n == 0.
