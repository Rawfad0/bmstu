# Тесты для лабораторной работы №03_04_01
## Входные данные
Целые m, n
Последовательность m*n целых чисел
## Выходные данные
Последовательность m целых чисел 
## Позитивные тесты:
- 01 - матрица 2x2 (все элементы);
- 02 - матрица 3x3;
- 03 - одинаковые числа в строках;
- 04 - матрица 4х4;
- 05 - все числа равны;
- 06 - матрица 1х1 (один элемент).
## Негативные тесты:
- 01 - пусто;
- 02 - не является числом (строка);
- 03 - только m;
- 04 - n == 0;
- 05 - m > 10;
- 06 - элементы не введены;
- 07 - количество элементов меньше заданного;
- 08 - матрица не квадратная.
