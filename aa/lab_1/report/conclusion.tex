\ssr{ЗАКЛЮЧЕНИЕ}

Цель работы достигнута: сравнение алгоритмов нахождения расстояния Левенштейна и Дамерау~---~Левенштейна было проведено.

В ходе выполнения лабораторной работы были решены все задачи:
\begin{enumerate}
	\item реализованы указанные алгоритмы поиска расстояния (два матричных, рекурсивный без кэширования и рекурсивный с кэшированием);
	\item проведены замеры затраченного процессорного времени выполнения реализованных алгоритмов;
	\item проведено исследование затрачиваемого процессорного времени и памяти при различных реализациях алгоритмов
	\item проведен сравнительный анализ реализаций алгоритмов по полученным  данным;
	\item описаны результаты в отчете.
\end{enumerate}

Сравнительный анализ реализаций показал:
\begin{enumerate}
	\item самый быстрый --- нерекурсивный алгоритм поиска расстояния Левенштейна;
	\item самый медленный ---  рекурсивный алгоритм поиска расстояния Левенштейна;
	\item нерекурсивный алгоритм поиска расстояния Левенштейна быстрее нерекурсивного алгоритма поиска расстояния Дамерау~---~Левенштейна;
	\item нерекурсивный алгоритм поиска расстояния Дамерау~---~Левенштейна быстрее рекурсивного алгоритма поиска расстояния Левенштейна с кэшированием;
	\item рекурсивный алгоритм поиска расстояния Левенштейна с кэшированием быстрее рекурсивного алгоритма поиска расстояния Левенштейна без кэширования.
\end{enumerate}
